# CLAWBOT
Simple Chat Bot

### For Linux User
```bash
$ apt update && apt upgrade
$ apt install git -y
$ apt install nodejs -y
$ apt install npm -y
$ git clone https://github.com/navetacandra24/ClawBot-WhatsApp
$ cd ClawBot-WhatsApp
$ npm install
```

###### Run
```bash
$ node .
```
